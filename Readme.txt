Blood Bank Management System 

For Database 
1. First open XAMPP server and start Apache and mySQL in it.
2. Then open phpmyAdmin.
3. Create a database name "login" 
4. Import the given "login.sql" file into it.


For Web Application User Interface
1. Go to the folder and open command propmt there and run "npm start". Then you can see the developent server starts in the browser.
2. Then open login/client and open another command promt and run "npm start".  Then browser will open in http://localhost:3000/ and display the web application.
